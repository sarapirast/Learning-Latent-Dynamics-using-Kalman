# Learning-Latent-Dynamics-using-Kalman

This repository contains the code for a project on learning linear
Gaussian state-space models from neural population activity.

## Features
- Sentence-wise neural preprocessing
- Gaussian smoothing and downsampling
- Kalman filtering and RTS smoothing
- PCA and random initialization
- Expectation-Maximization learning
- Reconstruction accuracy evaluation (NRMSE)

##Requirements
-numpy
-scipy
-matplotlib
-scikit-learn

## How to run
```bash
python main.py
